RANGKUMAN GIT 2 MINGGU LALU

Basic Workflow

  • git status: Cek status file
  • git add . : Stage semua perubahan
  • git commit -m "pesan" : Simpan perubahan
  • git push: Upload ke remote

Download atau Update

  • git pull: Download + merge perubahan terbaru
  • git fetch: Cek update tanpa merge

Branch management

  • git branch: Lihat semua branch
  • git branch namabranch: Buat branch baru
  • git checkout namabranch: Pindah branch
  • git switch namabranch: Alternatif pindah branch
  • git merge namabranch: Gabungkan branch

Remote management

  • git remote -v: Lihat semua remote
  • git remote add nama url: Tambah remote baru
  • git push nama-remote main: Push ke remote tertentu

Git Config

  • git config --global user.name "nama"
  • git config --global user.email "email"

Check History

  • git log: Lihat commit history
  • git log --oneline: History versi singkat

Undo/Fix

  • git restore file.txt: Batalkan perubahan file
  • git reset --soft HEAD~1: Batalkan commit terakhir
